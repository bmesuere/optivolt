Daily Accuracy Summary

How should we show per-day accuracy alongside the existing per-slot charts?

A

Daily bar chart below the diff chart

Add a third chart below each accuracy section: a bar chart with one bar per day showing total daily error (e.g. MAE or total absolute diff in kWh). Bars could be color-coded green→red by accuracy. Compact and keeps all accuracy info stacked vertically.

C

Grouped daily bar chart (actual vs predicted totals)

A grouped bar chart where each day has two bars side by side: total actual kWh and total predicted kWh. This directly shows how much energy was consumed/produced vs what was predicted per day. The visual gap between bars instantly communicates accuracy. An optional error % label on top of each pair.
